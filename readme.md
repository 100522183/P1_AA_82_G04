### Integrantes

### Descripción
Este proyecto predice si un cliente bancario suscribirá un depósito a plazo utilizando diferentes métodos de aprendizaje automático.

### Estructura del proyecto
- `notebooks/`: Contiene los notebooks de análisis y predicción
- `models/`: Modelo final entrenado
- `outputs/`: Predicciones generadas
- `app/`: Aplicación web con Streamlit
